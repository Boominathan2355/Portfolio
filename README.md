## Getting Started

### Prerequisites
- Node.js (v16+)
- npm (v8+)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Boominathan2355/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
